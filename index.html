foobar2000 fleshgrinder

This is my personal foobar2000 mod. I’m sharing this here mainly for my friends who like my configuration, but of course anybody is welcome to give it a shot. This new version incorporates some stuff that many people asked for. There is now support for the artreader functionality of foobar2000 and ratings are supported as well. My personal configuration that I’m using is a bit more stripped down for higher performance, but I keep this stuff available in the download version, so it’s hopefully easier to work with it for you guys.

Getting started

Please follow these steps carefully, otherwise it might not work for you. Please also note that I only had the chance to test my foobar2000 mod on Windows 7 and 8, it might work or might not work on older Windows operating systems.

Step 1

Download and install the free Open Sans font, this font is needed for this mod to work.

Step 2

You can skip this if you are not using Last.fm!

Enter your Last.fm username in the following text box and hit the “Generate Database” button. This will generate the Last.fm database for your foobar2000 installation.

Step 3

Download and unzip foobar2000. See the bottom of this page for alternate foobar2000 versions. This is the standard version that includes all functionalities.

Step 4

Start foobar2000 and set up your media library and enter your Last.fm credentials.

Click on the preferences icon.

Add your media library (e.g. the music folder of your Windows user).

Step 5

Enter your Last.fm credentials in the Audioscrobbler component settings and enable it.

Step 6

Enter your Last.fm credentials in the Soft Playlists component settings.

Step 7

Enter your Last.fm username in the WSH Panel Mod. To do so, right click the heart icon and select configure from the menu.

Enjoy!

That’s it, now you can create an autoplaylist or any other playlist and start listening to music.
